Техническое задание (ТЗ)
Платформа агрегации информации о университетах для международных абитуриентов
________________


1. ОБЩАЯ ИНФОРМАЦИЯ О ПРОЕКТЕ
1.1 Название проекта
Платформа по подбору университетов для международных студентов
1.2 Цель проекта
Создать веб-платформу, которая агрегирует детальную информацию о вузах (США, Европа, Великобритания, Китай, Гонконг и др.) и помогает абитуриентам из СНГ и Центральной Азии в процессе поступления через ИИ-ассистента, базу данных университетов и инструменты для планирования поступления.
1.3 Целевая аудитория
* Основная: Международные старшеклассники из СНГ и всего мира
* Вторичная: Родители абитуриентов
* Дополнительная: Репетиторы, образовательные консультанты, менторы
* B2B: Университетские офисы для партнёрств
1.4 Технические ограничения
* Платформа: Веб-сайт (адаптивный дизайн для мобильных устройств)
* Стек технологий: На усмотрение разработчика
* СУБД: На усмотрение разработчика
* Безопасность и масштабирование: На усмотрение разработчика
________________


2. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ (MVP)
2.1 Система аутентификации и управления пользователями
2.1.1 Типы пользователей
Система поддерживает три типа пользователей:
1. Абитуриент (Student) — основной пользователь платформы.
2. Консультант хаба (Internal Consultant) — сотрудник команды проекта, работающий с заявками студентов.
3. Администратор (Admin) — полный контроль над платформой, управление пользователями и контентом.

________________


2.1.2 Регистрация и авторизация
Для всех типов пользователей:
   * Регистрация через email + пароль.
   * Авторизация через OAuth (Google, Facebook).
   * Восстановление пароля через email.
   * Email-верификация после регистрации.
Особенности для консультантов хаба:
   * Консультант не регистрируется самостоятельно.
   * Аккаунт создаётся администратором.
   * Консультант получает доступ только к CRM для работы с заявками.

________________


2.1.3 Профиль абитуриента
Профиль должен содержать следующие поля:
Личная информация:
      * Имя, фамилия
      * Страна проживания
      * Город
      * Email, телефон (опционально)
      * Аватар (опционально)
Академические данные:
      * GPA (шкала 4.0 или процентная)
      * Результаты тестов:
      * SAT (общий балл, Math, Reading & Writing)
      * ACT (общий балл)
      * IELTS (общий балл + секции)
      * TOEFL (общий балл)
      * Год окончания школы / текущий класс
Внеклассная деятельность (Extracurriculars):
      * Список активностей (название, описание, длительность)
      * Достижения и награды
      * Волонтёрская деятельность
      * Лидерские позиции
Финансовая информация:
      * Максимальный годовой бюджет на обучение (USD)
      * Потребность в финансовой помощи (да/нет)
      * Типы стипендий, на которые претендует
Предпочтения:
      * Предпочитаемые страны (множественный выбор)
      * Интересующие специальности (majors) — множественный выбор
      * Предпочитаемый размер университета (маленький/средний/большой)
      * Желаемые программы (бакалавриат/магистратура/PhD)

Функционал профиля:
         * Редактирование всех полей
         * Список избранных университетов (добавление/удаление)
         * История взаимодействия с ИИ-ассистентом (опционально)

________________


2.1.4 Консультации (Internal Consultant) 
Описание:
            * Консультации проводятся только внутренней командой хаба.
            * Никаких внешних менторов и отдельной регистрации.
Функционал студента:
            * Оставляет заявку на консультацию через форму.
            * Указывает тему, предпочтительный формат (офис/онлайн) и удобное время.

Функционал консультанта хаба:
               * Консультант имеет доступ только к CRM-заявкам, поступившим от студентов.
               * Может просматривать:
               * данные студента (имя, контакты)
               * текст заявки
               * статус заявки
               * Не управляет оплатой (оплата напрямую в офисе или переводом).
Примечание по оплате:
               * Платформа не обрабатывает и не хранит платежные данные.
               * Платформа фиксирует только заявку и назначенного консультанта.
________________


2.2 База данных университетов
2.2.1 Структура данных университета
Для каждого университета должна храниться следующая информация:
Блок 1: Основная информация
               * Официальное название (на английском)
               * Название на русском (перевод)
               * Слоган (motto) — опционально
               * Страна
               * Город
               * Точный адрес
               * Координаты (latitude, longitude) для отображения на карте
               * Контакты приёмной комиссии:
               * Email
               * Телефон
               * Официальный сайт
               * Ссылка на страницу приёмной комиссии
               * Логотип (изображение)
               * Галерея фотографий кампуса (минимум 5 фото)
Блок 2: Академическая информация
               * Список факультетов (Schools/Colleges)
               * Список специальностей (Majors):
               * Название специальности
               * Описание (краткое)
               * Ссылка на страницу программы
               * Типы программ:
               * Бакалавриат (да/нет)
               * Магистратура (да/нет)
               * PhD (да/нет)
               * Язык обучения (английский/другой)
Блок 3: Требования к поступлению
Академические требования:
               * GPA:
               * Минимальный GPA
               * Средний GPA принятых студентов
               * Стандартизированные тесты:
               * SAT:
               * Требуется (да/нет/опционально)
               * Минимальный балл
               * Средний балл принятых (25th-75th percentile)
               * ACT:
               * Требуется (да/нет/опционально)
               * Минимальный балл
               * Средний балл принятых
               * IELTS:
               * Минимальный общий балл
               * Минимальные баллы по секциям (если есть)
               * TOEFL:
               * Минимальный балл (iBT)
               * Минимальные баллы по секциям (если есть)
Дополнительные требования:
               * Наличие собеседования (да/нет/опционально)
               * Требование мотивационного письма (Personal Statement) — да/нет
               * Supplemental essays:
               * Количество эссе
               * Темы (список)
               * Ограничение по количеству слов (word count)
               * Рекомендательные письма (количество)
               * Портфолио (для творческих специальностей) — да/нет
               * Другие требования (текстовое поле)
Блок 4: Системы подачи документов
               * Common Application:
               * Принимает (да/нет)
               * Ссылка на профиль университета в Common App
               * Coalition Application:
               * Принимает (да/нет)
               * Ссылка
               * Собственная система подачи:
               * Используется (да/нет)
               * Ссылка на application portal
               * Инструкции по подаче (текстовое поле с форматированием)
Блок 5: Финансовая информация
Стоимость обучения:
               * Tuition (стоимость обучения в год для международных студентов, USD)
               * Room & Board (проживание и питание в год, USD)
               * Прочие обязательные расходы (книги, страховка, транспорт и т.д., USD)
               * Общая стоимость за год (автоматически рассчитывается: tuition + room & board + прочие расходы)
Финансовая помощь:
               * CSS Profile требуется (да/нет)
               * Типы стипендий:
               * Merit-based scholarships:
               * Доступны (да/нет)
               * Описание
               * Критерии получения
               * Средний/максимальный размер (USD)
               * Need-based financial aid:
               * Доступна для международных студентов (да/нет)
               * Описание
               * Средний размер помощи
               * Full-ride scholarships:
               * Доступны (да/нет)
               * Название программы
               * Критерии отбора
               * Дедлайн
               * Процент студентов, получающих финансовую помощь
               * Дедлайны для подачи на финансовую помощь
Блок 6: Дедлайны
               * Early Action (EA):
               * Доступен (да/нет)
               * Дата дедлайна
               * Binding (обязывающий) — да/нет
               * Early Decision (ED):
               * Доступен (да/нет)
               * Дата дедлайна (ED I, ED II)
               * Обязательно binding
               * Regular Decision:
               * Дата дедлайна
               * Rolling Admission:
               * Доступен (да/нет)
               * Период приёма заявок
               * Дедлайны для международных студентов (если отличаются)
               * Дедлайны для стипендий (список с датами)
Блок 7: Статистика и показатели
Статистика поступления:
               * Acceptance rate (общий процент поступления, %)
               * Acceptance rate для международных студентов (%)
               * Средний GPA принятых студентов
               * Средние баллы SAT принятых (Math + EBRW)
               * Средние баллы ACT принятых
               * Yield rate (процент принявших предложение о зачислении)
Информация об университете:
               * Общее количество студентов
               * Количество иностранных студентов
               * Процент иностранных студентов
               * Соотношение студентов к преподавателям (student-to-faculty ratio)
               * Размер кампуса (акры/гектары)
Демографические данные:
               * Гендерное соотношение (мужчины/женщины, %)
               * Этническое разнообразие (опционально)
Карьерные показатели:
               * Employment rate (процент трудоустроенных через 6 месяцев после выпуска)
               * Средняя зарплата выпускников (первый год после выпуска, USD)
               * Топ-работодатели выпускников (список компаний)
Блок 8: Рейтинги
               * QS World University Ranking (позиция, год)
               * US News & World Report Ranking (позиция, год)
               * Times Higher Education Ranking (позиция, год)
               * Niche Grade (буквенная оценка: A+, A, B+ и т.д.)
               * Ссылки на источники рейтингов
Блок 9: Возможности и программы
Летние программы:[a]
               * Название программы
               * Описание
               * Даты проведения
               * Стоимость (USD)
               * Дедлайн подачи
               * Ссылка на страницу программы
Хакатоны:
               * Название
               * Даты
               * Требования к участникам
               * Призы
               * Ссылка
Исследовательские программы:
               * Название
               * Описание
               * Требования
               * Как подать заявку
Стажировки и партнёрства:
               * Программы стажировок для студентов
               * Партнёрские компании
               * Career services (описание)
Программы обмена:
               * Доступные программы международного обмена
               * Страны-партнёры
Блок 10: Локальная информация для Кыргызстана/СНГ
               * Представительства в регионе:
               * Наличие представительства в Кыргызстане/Казахстане (да/нет)
               * Адрес
               * Контакты
               * Локальные гранты и программы:
               * Название программы
               * Описание
               * Критерии
               * Как подать заявку
               * Программы подготовки в регионе
               * Контакты alumni (выпускников) в регионе
Блок 11: Дополнительный контент
               * Официальные ссылки:
               * Admission page
               * Financial aid page
               * International students page
               * Student life page
               * Campus tour page
               * FAQ (часто задаваемые вопросы) — текстовое поле
               * Виртуальные туры (ссылки на видео/360° туры)
               * Видео о университете (ссылки на YouTube и т.д.)
               * Описание студенческой жизни (клубы, организации, события)
2.2.3 Мультиязычность
               * Интерфейс платформы: полный перевод на русский и английский языки
               * Информация о университетах:
               * Названия университетов на обоих языках
               * Описания программ, требования — на английском (с возможностью добавления русского перевода в будущем)
               * Статические элементы интерфейса (кнопки, заголовки) — на обоих языках

2.3 Поиск и фильтрация университетов
2.3.1 Базовые фильтры (MVP)
Система поиска должна включать следующие фильтры:
               1. Страна (множественный выбор)

                  * Список стран: США, Великобритания, Канада, Германия, Нидерланды, Китай, Гонконг и другие
                  2. Специальность (Major) (множественный выбор)

                     * Список всех доступных специальностей в базе
                     * Группировка по категориям (Engineering, Business, Arts, Sciences и т.д.)
                     3. Требования по GPA (диапазон)

                        * Минимальный порог: от X до Y
                        * Показывать университеты, где требования соответствуют GPA пользователя
                        4. Требования по языковым тестам:

                           * IELTS: минимальный балл (от X до Y)
                           * TOEFL: минимальный балл (от X до Y)
                           * SAT: минимальный балл (от X до Y)
                           5. Наличие стипендий:

                              * Любые стипендии (да/нет)
                              * Merit-based (да/нет)
                              * Need-based (да/нет)
                              * Full-ride scholarships (да/нет)
                              6. Рейтинг (диапазон)

                                 * Фильтр по QS World Ranking: топ-50, топ-100, топ-200, топ-500
                                 * Фильтр по US News Ranking (аналогично)
2.3.2 Дополнительные фильтры (можно добавить позже)
                                 * Стоимость обучения (диапазон)
                                 * Acceptance rate (диапазон)
                                 * Размер университета (маленький/средний/большой)
                                 * Расположение (город/пригород/сельская местность)
                                 * Тип университета (Public/Private)
2.3.3 Универсальный поиск
                                 * Поиск по названию университета
                                 * Поиск по городу
                                 * Быстрый поиск с автодополнением
2.3.4 Функционал результатов поиска
                                 * Отображение результатов в виде карточек (grid view)
                                 * Отображение результатов списком (list view)
                                 * Сортировка результатов:
                                 * По рейтингу (убывание)
                                 * По стоимости (возрастание/убывание)
                                 * По acceptance rate (возрастание)
                                 * По fit-скору (если пользователь залогинен)
                                 * Pagination или бесконечная прокрутка
                                 * Показывать общее количество найденных университетов
                                 * Возможность добавить университет в избранное прямо из результатов поиска

2.4 Карточка университета
2.4.1 Структура страницы
Страница университета должна отображать всю информацию из базы данных, структурированную по разделам:
Шапка страницы:
                                 * Логотип университета
                                 * Название (на выбранном языке)
                                 * Страна, город
                                 * Кнопка "Добавить в избранное" / "Удалить из избранного"
                                 * Кнопка "Рассчитать fit-скор" (если пользователь залогинен)
                                 * Галерея фотографий кампуса (слайдер)
Навигация по разделам (tabs или якорные ссылки):
                                 1. Overview (Обзор)
                                 2. Admissions (Поступление)
                                 3. Academics (Программы обучения)
                                 4. Tuition & Aid (Стоимость и финансовая помощь)
                                 5. Statistics (Статистика)
                                 6. Opportunities (Возможности)
                                 7. Contact (Контакты)
Раздел 1: Overview
                                 * Краткое описание университета
                                 * Слоган (motto)
                                 * Рейтинги (QS, US News, THE, Niche) — с иконками
                                 * Карта с местоположением университета (встроенная)
                                 * Основные факты:
                                 * Год основания
                                 * Тип (Public/Private)
                                 * Количество студентов
                                 * Student-to-faculty ratio
                                 * Acceptance rate
Раздел 2: Admissions
                                 * Требования к поступлению (GPA, SAT, ACT, IELTS, TOEFL)
                                 * Дополнительные требования (эссе, рекомендации, портфолио)
                                 * Системы подачи (Common App, Coalition, собственная) с кнопками-ссылками
                                 * Дедлайны (EA, ED, Regular, Rolling) в виде таблицы или timeline
                                 * Инструкции по подаче
Раздел 3: Academics
                                 * Список факультетов
                                 * Список специальностей (majors) с описаниями
                                 * Типы программ (бакалавриат, магистратура, PhD)
                                 * Популярные программы
                                 * Особенности академической системы
Раздел 4: Tuition & Aid
                                 * Стоимость обучения (Tuition)
                                 * Стоимость проживания (Room & Board)
                                 * Прочие расходы
                                 * Общая стоимость за год (выделено жирным)
                                 * Доступные стипендии:
                                 * Merit-based
                                 * Need-based
                                 * Full-ride scholarships
                                 * Требование CSS Profile
                                 * Дедлайны для финансовой помощи
                                 * Процент студентов, получающих помощь
Раздел 5: Statistics
                                 * Acceptance rate (общий и для международных студентов)
                                 * Средние баллы принятых студентов (GPA, SAT, ACT)
                                 * Yield rate
                                 * Демография (гендерное соотношение, diversity)
                                 * Employment rate выпускников
                                 * Средняя зарплата выпускников
                                 * Топ-работодатели
Раздел 6: Opportunities
                                 * Летние программы (название, даты, стоимость, дедлайн)
                                 * Хакатоны
                                 * Исследовательские программы
                                 * Стажировки
                                 * Программы обмена
                                 * Локальные возможности (для Кыргызстана/СНГ)
Раздел 7: Contact
                                 * Email приёмной комиссии
                                 * Телефон
                                 * Адрес
                                 * Официальные ссылки (admission page, financial aid, international students)
                                 * FAQ
                                 * Виртуальные туры и видео
                                 * Контакты представительства в регионе (если есть)
                                 * Контакты alumni в регионе
2.4.2 Дополнительный функционал на странице
                                 * Кнопка "Apply Now" — ведёт на application portal университета

                                 * Кнопка "Share" — поделиться ссылкой в соцсетях

2.5 Fit-скор (оценка соответствия)
2.5.1 Описание функции
Fit-скор — это автоматически рассчитываемая оценка того, насколько университет подходит конкретному студенту на основе его профиля.
2.5.2 Параметры расчёта
Скор рассчитывается на основе следующих факторов:
                                 1. Академические показатели (40% веса):

                                    * Соответствие GPA студента требованиям университета
                                    * Соответствие SAT/ACT студента средним баллам принятых
                                    * Соответствие IELTS/TOEFL минимальным требованиям
                                    2. Внеклассная деятельность (30% веса):

                                       * Количество и качество extracurriculars
                                       * Лидерские позиции
                                       * Достижения и награды
                                       * Волонтёрство
                                       3. Финансовая совместимость (30% веса):

                                          * Соответствие бюджета студента стоимости обучения
                                          * Наличие подходящих стипендий (merit/need-based)
                                          * Вероятность получения финансовой помощи
2.5.3 Отображение результата
                                          * Процентное соотношение: например, "Your fit: 78%"
                                          * Цветовая индикация:
                                          * 0-40%: Красный (Low Match / Reach)
                                          * 41-70%: Жёлтый (Medium Match / Target)
                                          * 71-100%: Зелёный (High Match / Safety)
                                          * Категории:
                                          * Reach (0-40%): университет является очень амбициозным выбором
                                          * Target (41-70%): реалистичные шансы поступления
                                          * Safety (71-100%): высокие шансы поступления
2.5.4 Пояснения "почему такой скор"
Обязательное требование: система должна объяснять, почему был выставлен такой скор.
Формат пояснения:
Your Fit Score: 68% (Target)


Strengths:
✓ Your GPA (3.8) exceeds the average (3.5)
✓ Your IELTS score (7.5) meets requirements (min 6.5)
✓ Strong extracurricular profile matches university's values


Areas for improvement:
⚠ Your SAT score (1320) is below the average (1450)
⚠ Limited financial aid available for international students


Recommendation:
This university is a realistic target for you. Consider retaking the SAT to improve your chances, or emphasize your extracurriculars in your application essays.


2.5.5 Требования к реализации
                                          * Скор рассчитывается только для залогиненных пользователей с заполненным профилем
                                          * Кнопка "Рассчитать fit-скор" на карточке университета
                                          * Результат отображается модальным окном или отдельной секцией
                                          * Возможность пересчитать скор после обновления профиля
                                          * Сохранение истории расчётов (опционально)

2.6 База успешных подач (AdmitSee аналог)
2.6.1 Описание функции
Раздел платформы, где размещены профили студентов, успешно поступивших в университеты, с их реальными эссе, CV и другими материалами.
2.6.2 Способ наполнения базы
Вариант Б (выбран): Менторы и администраторы платформы находят студентов и договариваются с ними о размещении их профилей.
Процесс:
                                          1. Ментор/администратор связывается со студентом
                                          2. Студент даёт согласие на публикацию материалов
                                          3. Ментор/администратор загружает профиль через админ-панель
                                          4. Студент получает доступ к премиум-функциям платформы в качестве благодарности (опционально)
2.6.3 Структура профиля успешного студента
Основная информация:
                                          * Имя студента (если согласен) или "Anonymous Student"
                                          * Фото (опционально, если согласен)
                                          * Университет, в который поступил
                                          * Специальность (major)
                                          * Год поступления
                                          * Страна происхождения студента
Академические показатели:
                                          * GPA
                                          * SAT / ACT (баллы)
                                          * IELTS / TOEFL (баллы)
                                          * Класс окончания школы
Внеклассная деятельность (Extracurriculars):
                                          * Список основных активностей, которые упоминались в заявке
                                          * Достижения и награды
                                          * Лидерские позиции
                                          * Волонтёрство
Эссе:
                                          * Common App Essay (если применимо)
                                          * Supplemental Essays (по количеству, требуемому университетом)
                                          * Важно: публикация эссе только с явного согласия студента
Дополнительные материалы:
                                          * CV/Resume (если применимо)
                                          * Список курсов (если релевантно)
                                          * Рекомендации (краткое описание, не сами письма)
Результат:
                                          * Acceptance (поступил)
                                          * Financial aid received (размер полученной финансовой помощи)
                                          * Отзыв студента о процессе поступления (опционально)
2.6.4 Функционал раздела
Поиск и фильтрация:
                                          * Поиск по университету
                                          * Фильтр по специальности (major)
                                          * Фильтр по стране происхождения студента
                                          * Фильтр по диапазону GPA
                                          * Фильтр по диапазону SAT/ACT
                                          * Фильтр по году поступления
Отображение профилей:
                                          * Список профилей в виде карточек
                                          * Клик на карточку открывает полный профиль студента
                                          * Возможность добавить профиль в избранное
Страница профиля студента:
                                          * Вся информация, указанная в структуре выше
                                          * Кнопка "Download Essay" (скачать эссе в PDF/TXT)
                                          * Кнопка "Similar Profiles" (показать похожие профили)
2.6.5 Модерация и конфиденциальность
                                          * Все профили проходят модерацию администратором перед публикацией
                                          * Эссе публикуются только с явного письменного согласия студента
                                          * Студент может запросить удаление своего профиля в любой момент
                                          * Персональные данные (контакты) студентов не публикуются

2.9 Система менторства
2.9.1 Описание функции
Раздел платформы, где студенты могут найти менторов (образовательных консультантов) и записаться на консультацию.
2.9.2 Как это работает
                                          1. Студент просматривает список доступных менторов
                                          2. Выбирает ментора на основе профиля, опыта, специализации
                                          3. Оставляет заявку на консультацию через платформу
                                          4. Ментор получает уведомление о заявке (email или в личном кабинете)
                                          5. Ментор связывается со студентом напрямую
                                          6. Оплата производится напрямую ментору (студент приходит в офис)
Важно: Платформа только передаёт заявки. Никакой встроенной системы оплаты или видеосвязи не требуется.
2.9.3 Форма заявки на консультацию
После клика на кнопку "Записаться" открывается форма:
Поля формы:
                                          * Имя студента (автозаполнение, если залогинен)
                                          * Email
                                          * Телефон
                                          * Предпочитаемое время консультации (date picker)
                                          * Тип услуги (выбор из списка услуг ментора)
                                          * Комментарий / вопросы к ментору (текстовое поле)[b]
После отправки:
                                          * Студент видит сообщение: "Ваша заявка отправлена. Ментор свяжется с вами в течение 24 часов."
                                          * Ментор получает email с деталями заявки
                                          * Заявка появляется в личном кабинете ментора
2.9.4 Личный кабинет ментора
Раздел "Заявки":
                                          * Список всех входящих заявок
                                          * Статусы заявок:
                                          * Новая (new)
                                          * В обработке (in progress)
                                          * Завершена (completed)
                                          * Отклонена (declined)
                                          * Возможность изменить статус заявки
                                          * Контактная информация студента (email, телефон)
Раздел "Календарь":
                                          * Ментор может указать дни и часы, когда доступен для консультаций
                                          * Блокировка занятых слотов (опционально)
Раздел "Профиль":
                                          * Редактирование публичного профиля
                                          * Изменение стоимости услуг
                                          * Загрузка фото

2.10.1 Описание функции

2.13 Сравнение университетов (University Comparison)
2.13.1 Описание функции
Инструмент для side-by-side сравнения нескольких университетов по ключевым параметрам.
2.13.2 Как это работает
                                          1. Пользователь добавляет университеты для сравнения (максимум 3-4 одновременно)
                                          2. Система отображает таблицу с параметрами университетов
                                          3. Пользователь может экспортировать сравнение (PDF / Excel)
2.13.3 Параметры для сравнения
Основная информация:
                                          * Название, логотип
                                          * Страна, город
                                          * Рейтинги (QS, US News, THE)
Академические требования:
                                          * Минимальный GPA
                                          * SAT / ACT (средний балл принятых)
                                          * IELTS / TOEFL (минимум)
Финансы:
                                          * Tuition
                                          * Room & Board
                                          * Общая стоимость за год
                                          * Наличие merit / need-based / full-ride scholarships
Статистика:
                                          * Acceptance rate
                                          * Количество студентов
                                          * Student-to-faculty ratio
                                          * Employment rate выпускников
Дедлайны:
                                          * Early Action / Early Decision
                                          * Regular Decision
Доступные специальности:
                                          * Список majors (показывать только те, которые есть во всех сравниваемых университетах, с отметкой "✓" или "✗")
2.13.4 Интерфейс сравнения
Добавление университетов:
                                          * Кнопка "Compare" на карточке университета / странице университета
                                          * Боковая панель с выбранными университетами для сравнения
                                          * Возможность удалить университет из сравнения
Таблица сравнения:
                                          * Университеты в колонках (горизонтальная прокрутка для 3+ университетов)
                                          * Параметры в строках
                                          * Цветовое выделение лучших/худших значений:
                                          * Зелёный: лучшее значение (например, самый высокий acceptance rate среди safety schools)
                                          * Красный: худшее значение (например, самая высокая стоимость)

2.15 Административная панель
2.15.1 Основные функции
Управление университетами:
                                          * Добавление нового университета (форма со всеми полями из структуры 2.2)

                                          * Список заявок на верификацию менторов (новые / одобренные / отклонённые)

                                          * Снятие верификации с ментора (в случае нарушений)

                                          * Добавление новых гайдов (WYSIWYG редактор)

                                          * Удаление гайдов
                                          * Управление категориями гайдов

                                          * Добавление новых программ
                                          * Редактирование и удаление программ

Управление базой успешных подач:
                                          * Добавление новых профилей студентов
                                          * Редактирование профилей
                                          * Модерация (одобрение/отклонение) профилей перед публикацией
                                          * Удаление профилей по запросу студентов

                                          * Общее количество пользователей (по типам)

                                          * Популярные университеты (по просмотрам, избранным)

                                          * Статистика использования ИИ-ассистента (количество запросов, популярные темы)
                                          * География пользователей
                                          * Конверсия (регистрация → активация → premium подписка)
Система уведомлений (опционально):
                                          * Отправка массовых email пользователям

                                          * История отправленных уведомлений
